Baxter the Barista

About the Project

This was my team's entry into the final competition for ME495: Embedded Systems in Robotics at Northwestern. The format of the project was very open-ended, with teams choosing demonstrations ranging from setting a table to playing a game of checkers using Rethink's Baxter robot.

Take a look at the README for information on how it works, and download the ROS package from GitHub to try it out yourself (Baxter required!).

Skills Involved

  • ROS (including custom messages & services)
  • Python
  • Inverse Kinematics & motion planning
  • Image processing and object detection with OpenCV
  • AR tag tracking & identification
  • Collaborative GitHub usage
